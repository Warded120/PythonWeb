# Project Overview

## Entities

- **User** → name, email, password, type (individual / business)  
- **Product** → name, category, price, stock quantity  
- **Order** → user, list of products, status  
- **Delivery** → address, date  
- **Cart** → temporary storage of selected products  

## Main Use Cases

- User browses products  
- Adds them to the cart  
- Places an order  

## Project Implementation Stages

**Stage 1** → Static landing page (HTML + CSS)  
**Stage 2** → Simple backend (Python Django), database setup  
**Stage 3** → Implementation of cart and order functionality  
**Stage 4** → User authentication and role separation (User/Admin)  
**Stage 5** → Integration of payments and delivery
